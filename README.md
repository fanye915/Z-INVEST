# Z-INVEST 投资组合追踪
这是一个基于新浪财经免费接口的投资组合追踪网站，支持A股、港股、美股的基金实时价格查询和收益计算。

## 功能
- 实时获取基金的最新价格
- 计算每个基金的持仓收益和总收益
- 展示收益走势图表
- 支持手动刷新数据

## 部署方法
1. Fork 本仓库
2. 上传 `index.html` 到你的GitHub仓库
3. 开启GitHub Pages功能，选择该仓库的主分支作为源
4. 访问你的GitHub Pages地址即可使用

## 使用说明
- 页面加载后会自动获取最新的基金数据
- 点击每个组合的"刷新最新数据"按钮可以手动刷新数据
- 收益走势图表会展示总成本和当前价值的变化趋势

## 注意事项
- 新浪财经的接口有15分钟左右的延迟，这是免费接口的正常限制
- 如果需要更实时的数据，可以考虑使用新浪的开放平台接口，但需要申请token
- 本网站仅用于个人学习和使用，请勿用于商业用途
